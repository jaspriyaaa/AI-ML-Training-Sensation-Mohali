## 📌 Overview

On Day 11, we trained our first Machine Learning model using the preprocessed house pricing dataset. The dataset had already been cleaned in the previous session through missing value handling, outlier treatment, feature scaling, and categorical encoding.

The goal of this training was to build a **Linear Regression Model** capable of predicting house prices based on different property features.

### What is Linear Regression?

Linear Regression is a supervised machine learning algorithm used to predict continuous numerical values. It identifies the relationship between input features and the target variable by fitting the best possible straight line through the data.

The mathematical equation of Linear Regression is:

[
y = mx + c
]

Where:

* **y** = Predicted value
* **x** = Input feature
* **m** = Slope (Coefficient)
* **c** = Intercept

In this project, the model learns how different house characteristics influence the final house price.

# 🎯 Step 8: How Do We Know If Our Model Is Accurate? (Model Evaluation)

### Objective

Measure model performance using evaluation metrics.

---

## 1️⃣ Mean Absolute Error (MAE) 💸

### Definition

MAE measures the average difference between actual and predicted prices.

### Formula

[
MAE = \frac{\sum |Actual - Predicted|}{n}
]

### Interpretation

* Lower MAE = Better model
* Higher MAE = Larger prediction errors

### Purpose

Shows the average amount of money by which the model is wrong.

---

## 2️⃣ R² Score (Coefficient of Determination) 📈

### Definition

Measures how much variation in house prices is explained by the model.

### Range

* **1.0** = Perfect prediction
* **0.0** = No predictive power

### Interpretation

Higher R² values indicate stronger predictive performance.

### Purpose

Evaluates how effectively the model captures relationships in the dataset.
